# SafeRoute

# Problem Statement
Women and travelers often face safety concerns while traveling at night or through unknown areas. 
SafeRoute helps users find the safest route by considering safety scores, nearby safe places, emergency alerts, and night mode assistance.

# Data Structures Used
- Graph → to represent locations and routes
- Priority Queue → used in Dijkstra algorithm
- Queue → used in BFS for nearest safe place search
- ArrayList / HashMap → storing nodes, routes, and trip history

# Project Features
- Safest route finder
- Emergency alert system
- Nearest safe place detection
- Night mode support
- Trip history logging
